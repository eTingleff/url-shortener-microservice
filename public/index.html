URL Shortener Microservice

A Free Code Camp API Project Created by Eric Tingleff

Instructions


Shorten a URL:

Add the following to the end of this page's URL:

/shorten/<URL-to-be-shortened>

Note:

  • URLs for non existant websites will not be shortened
  • URLs must include an 'http://' or 'https://' prefix


Example


To shorten https://youtube.com

Navigate to https://url-shortener-microservice-fcc.glitch.me/shorten/https://www.youtube.com
Example Response:
Original URL: https://www.youtube.com,
Shortened URL: https://url-shortener-microservice-fcc.glitch.me/4028

Navigating to the shortened URL should direct to https://www.youtube.com
